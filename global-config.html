# 新增配置

可以在测试用例中的测试步骤的请求URL地址，json参数，查询字符串参数，x-www-form-urlencoded参数，请求头参数，cookies参数中使用$全局变量名的方式来使用全局变量。运行用例时，系统后台会自动解析以上测试步骤中的请求参数中引用了全局变量的参数，然后将测试步骤请求参数中的$全局变量名替换为对应的全局变量值。具体使用详情请查看测试用例管理模块用户文档。
