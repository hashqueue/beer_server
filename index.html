# Beer接口自动化测试平台(单机版项目)

# 预览

# 使用到的技术栈

# 前端

  • Vue.js (opens new window) JavaScript框架
  • Vuex (opens new window) 数据集中式存储；跨组件通信
  • Vue Router (opens new window) Vue.js 官方的路由管理器。
  • Ant Design of Vue (opens new window) UI组件库
  • G2Plot (opens new window) 开箱即用的图表库
  • axios (opens new window) 易用、简洁且高效的基于Promise的http库
  • Monaco Editor (opens new window) 实现在线编辑器功能，在线编辑Python代码以及渲染JSON

# 后端

  • Django (opens new window) Python Web 框架
  • Django REST framework (opens new window) 基于Django构建restful风格的API
  • djangorestframework-simplejwt (opens new window) 用于Django REST framework的JSON Web令牌身份验证插件
  • drf-spectacular (opens new window) 生成在线swagger和redoc形式的API文档
  • Celery (opens new window) 实现异步任务队列功能
  • django-celery-results (opens new window) 使用Django ORM存储Celery任务执行结果
  • django-cors-headers (opens new window) 后端配置CORS解决跨域问题
  • django-grappelli (opens new window) Django自带admin后台的主题美化插件
  • Gunicorn (opens new window) 用于生产环境部署Django项目的Python WSGI HTTP Server
  • jmespath (opens new window) 用于JSON查询的语言，实现接口自动化测试中的响应体结果的提取
  • Mysql (opens new window) 项目数据持久化存储
  • RabbitMQ (opens new window) 用于Celery异步任务队列的消息中间件
  • Requests (opens new window) 使用Python发起HTTP请求
  • Nginx (opens new window) Web和反向代理服务器
  • Docker (opens new window) 容器化部署前后端应用
  • Docker Compose (opens new window) 容器编排工具，实现一键部署项目

# 特别鸣谢以上开源项目，让我得以站在巨人的肩膀上快速构建自己的项目。

# 后端开发

  1. 在compose/server目录下的config.ini配置文件中修改测试环境所需要的环境配置(数据库需要自己手动创建)
  2. 返回到项目根目录安装项目依赖库，然后进行数据迁移(会自动在上一步中创建的数据库中创建开发所需要的数据表)
    # 返回项目根目录
cd .../../
# 安装项目依赖库
pip3 install -i https://pypi.doubanio.com/simple -U pip && pip3 install -i https://pypi.doubanio.com/simple -r requirements.txt
# 创建迁移记录
python3 manage.py makemigrations
# 数据迁移(创建数据库表)
python3 manage.py migrate

    # 创建rabbitmq容器

# 启动Celery服务(在项目manage.py文件同级目录下用该命令启动)
